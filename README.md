# Rock Paper Scissors Game

A Java console game where you play Rock, Paper, Scissors against the computer.

## What This Program Does

- Play Rock, Paper, Scissors against a computer opponent
- Computer makes random choices using Java's Random class
- Accepts both shortcuts (r/p/s) and full words
- Determines winner based on standard game rules

## How to Run

1. Make sure you have Java installed
2. Open the project in your code editor
3. Run the `RockPaperScissorsGame.java` file
4. Follow the on-screen instructions

## Sample Usage

Rock Paper Scissors Game
To make things easier, type:
r for rock, p for paper, s for scissors
Your choice: r
You chose: rock
Computer chose: paper
Paper covers rock. Computer wins!
